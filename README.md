## Overview
**DualSenseX DSX Steam Edition** is an enhanced version of DualSenseX that provides full support for PlayStation 5 DualSense controller features on Windows, with excellent Steam integration. Enjoy adaptive triggers, haptic feedback, motion controls, LED customization, and more.

> [!Important]
> This tool is intended for use with legitimate DualSense controllers and Steam games.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `DualSenseX.exe` as Administrator
3. Connect your DualSense controller via USB or Bluetooth
4. Configure settings and enjoy enhanced gameplay

---
## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| Controller not detected    | Check USB/Bluetooth connection                |
| Features not working       | Make sure Steam is running                    |
| Antivirus warning          | Add the folder to exclusions                  |
| High latency               | Use wired USB connection instead of Bluetooth |
